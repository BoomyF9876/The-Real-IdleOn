# The Real IdleOn
A three-day game-jam project for Tier-9 studio take-home assignment

## Where's The Game
```./Build/The Real IdleOn.exe```

## How To Play
Mouse Clicks Only!

## Game Introduction
Similar to IdleOn, this game is a 2D platformer game. However, based on the limited timeline, this game discards the MMO element and focuses entirely on the idle/auto-battler elements. Key design pillars are:

### Auto Battle
In this game, the character will always find the nearest enemy and start auto fighting. No player control, no dash or dodge, just straight-up fighting.

### Character Progression
The key mechanics in this game is to collect the coins from enemies to upgrade your character and prepare for stronger enemies. The upgrade options are: health, health regen, movement speed, attack damage, attack speed, +coins on enemies, +%chance to block enemy attack, etc.

### Infinite Playability
Similar to other idle games, the idea is to make sure the game has infinite progression loop. The game content may be repeated in late game, but the player can still feel the progression on the character and has momentum to keep playing.
